The 8 Specialized Agents

Agent 0: Root Orchestrator

The boss. Coordinates all sub-agents via A2A Protocol to ensure smooth daily workflow execution.

Agent 1: Topic Manager

Manages tracked topics and keywords. Reads/writes to Firestore topics_to_monitor collection.

Agent 2: News Aggregator

Fetches news from RSS feeds and APIs. Collects 100+ articles daily from configured sources.

Agent 3: Relevance Scorer

Scores articles for importance using custom algorithms. Outputs relevance scores (1-10).

Agent 4: Article Analyst

AI-powered analysis using Gemini 2.0 Flash. Generates summaries, tags, and strategic insights.

Agent 5: Daily Synthesizer

Creates executive briefs from analyzed articles. Delivers "what actually matters" each day.

Agent 6: Validator

Ensures data quality through schema validation. Keeps everything clean and consistent.

Agent 7: Storage Manager

Saves data to Firestore and BigQuery. Manages articles, daily_summaries, and metrics.
